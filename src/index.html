Frequently Asked Questions

How can I upload content?

Use the form above to upload your content. For skins or coverflows, provide the official forum link and a screenshot.

I've uploaded content and it won't appear, how is it?

The content is approved within 12-72 hours where the admin checks that the files and the source website are valid.

It has passed more than 72 hours since I uploaded the file, what happened?

Maybe the file you supplied doesn't work, the URL you give isn't from a forum, or it's inappropriate. Contact the admin to consult why it hasn't been approved yet.

The script showed me an error / The script won't work for me. What can I do?

First of all, update the script if possible. If it keeps failing, contact the admin with the debug.log file from your Aurora folder.
